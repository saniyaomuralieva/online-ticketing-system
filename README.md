# Online Ticketing System Lab

This is lab work for the online ticketing system.

In this project, I added a small Python function that converts Celsius to Fahrenheit. I also added unit tests to check if the function works correctly.

## Files

- `temperature.py` has the Celsius to Fahrenheit function.
- `test_temperature.py` has the tests for the function.

## Formula

Fahrenheit = Celsius * 9/5 + 32
